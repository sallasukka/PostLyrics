# PostLyrics

Do you remember how we used to post vague, emotional song lyrics on Microsoft messenger status? I'm at that stage of my life now, again. So this is an app for posting emotional lyrics for other to see (bring emo back).

Features: 
- Personal profile (I want to know minimal info about users for security reasons)
- Feed (you see others' lyric posts; you can like and comment on them)
- Posting lyrics (minimal template; you can choose only the background color in the early versions of this app and a couple of fonts) 

Technologies used:
- Expo
- React Native
- Supabase
- Typescript
